# 🛒 E-Commerce Website

## 📖 Project Description

The **E-Commerce Website** is a full-stack web application developed using **Django** that provides a seamless online shopping experience. It allows users to browse products, view detailed product information, add items to a shopping cart, and place orders through an intuitive and responsive interface.

The application also includes an **admin dashboard** where administrators can efficiently manage product categories, inventory, and customer orders. The project demonstrates the implementation of Django's MVC architecture, user authentication, database management, and dynamic web development.

---

## ✨ Key Features

* 👤 User Registration & Secure Login
* 🛍️ Browse Products by Category
* 🔍 Product Details Page
* 🛒 Shopping Cart Management
* 📦 Order Placement
* 👨‍💼 Admin Dashboard
* 📱 Responsive User Interface
* 💾 SQLite Database Integration

## 🚀 Getting Started

### Clone the Repository
git clone <repository-url>

### Navigate to the Project
cd ecommerce

### Install Dependencies
pip install -r requirements.txt

### Run the Development Server
python manage.py migrate
python manage.py runserver


Open your browser and visit:
http://127.0.0.1:8000/
