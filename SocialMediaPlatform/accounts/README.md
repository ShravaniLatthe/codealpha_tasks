# 🌐 Social Media Platform

## 📖 Project Description

The **Social Media Platform** is a Django-based web application designed to simulate the core functionality of a modern social networking website. The platform enables users to create accounts, build personal profiles, share posts, and engage with content in an interactive environment.

The project focuses on implementing user authentication, content management, and social interaction features while following Django's best practices for scalable and maintainable web applications.


## ✨ Key Features

* 🔐 User Registration & Login
* 👤 User Profile Management
* 📝 Create, Edit & Delete Posts
* ❤️ Like Posts
* 💬 Comment on Posts
* 📰 Dynamic News Feed
* 🖼️ Image Upload Support
* 👨‍💼 Django Admin Panel
* 📱 Responsive User Interface

## 🚀 Getting Started

### Clone the Repository
git clone <repository-url>

### Navigate to the Project
cd SocialMediaPlatform

### Install Dependencies
pip install -r requirements.txt

### Run the Development Server
python manage.py migrate
python manage.py runserver

Open your browser and visit:
http://127.0.0.1:8000/
